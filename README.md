# 💬 DevChat — Real-Time Chat Application

A full-stack real-time messaging application built using the MERN stack and Socket.io.

---

## 🚀 Features

- Real-time messaging
- Room-based chat
- User authentication
- Multiple users support
- Responsive UI

---

## 🛠️ Tech Stack

Frontend:
- React.js
- HTML, CSS

Backend:
- Node.js
- Express.js
- Socket.io

Database:
- MySQL
